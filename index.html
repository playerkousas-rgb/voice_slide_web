Voice Cue Slide Tool

GitHub + Vercel Static MVP

測試版功能

  • 本機載入投影片圖片
  • 載入 / 下載 project.vcue.json
  • Cue Builder 設定觸發與動作
  • Chrome / Edge 語音辨識
  • 全螢幕播放器 + 人手後備

Google Drive 模式

第一版先用 Google Drive 資料夾手動工作流：把 slides、司儀稿、project.vcue.json 放同一資料夾分享。真正 Drive API 放第二階段。

選擇使用方式

先選純瀏覽器即插即用，或 Google Drive 資料夾工作流。

已有 project.vcue.json？

直接載入即可恢復 cue、模式、門檻、冷卻、司儀稿和 slide title。Browser 模式下，再選一次投影片圖片資料夾即可。

A 檔：投影片

PowerPoint / Canva / Google Slides 建議先匯出成 PNG/JPG。選擇整個資料夾，系統會按檔名排序。

Slides
選擇或拖放投影片圖片

PowerPoint 匯出通常是 Slide1.PNG、Slide2.PNG……；Canva 下載 zip 後請先解壓。你也可以將 PNG/JPG 檔案或整個資料夾拖到這裡。

Project / Cue 匯入

Project JSON 保存完整設定；CSV 可快速匯入傳統 cue 表。

Browser 安全限制：Project JSON 不會保存圖片檔案本身；重用時需要重新選一次 slides 資料夾。Google Drive API 第二版可改善。

B 檔：司儀稿 / 逐字稿

第一版支援 TXT 或直接貼上。DOCX/PDF 可先複製文字貼上；之後再做解析。

Script

Slide Titles

「參閱 XXX」模式會用這些標題做 matching。可由檔名產生，再手動修改。

安全與語音設定

正式活動建議：Sequential + 長而獨特的 cue + 人手鍵盤後備。若現場太嘈，可改用 Confirm 模式。

Cue Builder

設定「聽到甚麼」和「執行甚麼動作」。Cue 表是真正放映時會使用的規則。

#啟用觸發字眼 / 句子動作目標門檻工具

從司儀稿加入 Cue

點一句即可加入 cue。目標頁可之後在表格修改。
